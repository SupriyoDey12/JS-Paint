About Js Paint.

  • what is js paint?

Js Paint is a clone application of Microsoft Paint. But the basic difference is that, it is a web based application which is created with javaScript.

  • why use js paint?

It can be used from any operating system and also from android mobile. If you are logged in then your drawing will be saved in our database and you can download it anytime from anywhere. You can use every single features of Microsoft Paint here.

  • who builded this js paint?

We are a team of six computer science students, have built this site as a project work.

  • how to use it?

Just access the site link from where you want to access js paint (eg. Laptop, Desktop). Then click on Explore Paint or paint from navigation menu.
